作用域、沙箱、闭包

作用域

  • 变量起作用的区域
  • 作用域链：在一个函数内部嵌套另外一个函数
    注：不管在哪里调用，只管在哪里创建

闭包

  • 定义：在函数内部定义的变量，在函数外部无法访问
  1. js中所有的函数都能形成闭包
  2. 函数里面包裹其他函数，并且返回这个函数

    function fn () {
    var num = 123;
    return function () {

}
}
var n = fn();//结果：return 后面的函数体
n();
