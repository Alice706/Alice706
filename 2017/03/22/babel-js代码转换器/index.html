vue起航2 (babel)

babel js代码转换器

可以用来转换ES6/ES7/react… =>ES5的代码
配合webpack一起使用

  • 缺点：
    • 使用babel默认只解析关键字，不处理函数
    • 解决方案： 设置 ES6,插件：转换函数
  • 创建配置文件：
    • .babelrc文件
    • 数据是一个Json对象
      • {“presets(预设)”:[‘es2015’],”plugins(插件)”:[‘transform-runtime’]}

package.json中的代码

  • dependencies：项目生产所依赖的模块
  • devDependencies：项目开发所需要的模块
  • –save参数表示将该模块写入dependencies属性，
  • –save-dev表示将该模块写入devDependencies属性。

常用指令

注： v-bind:class可以简写为 :class
v-bind是给属性赋值
v-on:click=”*“可以简写为: @click
v-on是给事件做绑定，配合methods使用
