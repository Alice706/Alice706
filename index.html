创建播客遇到的一些问题

  1. 电脑上需要安装 node.js 和 git

  2. 在git bash中通过 npm install hexo-cli -g下载hexo
    注：如果npm比较慢，可以先安装淘宝的cnpm
    npm install -g cnpm –registry=https://registry.npm.taobao.org
    具体的建议先百度一下，防止这个语法已经变更了

  3. hexo init blog 通过这个命令初始化
    我在安装时遇到一个坑在 hexo init最后会出现一个乱码，然后提示:
    npm WARN deprecated swing@1.4.2: This package is no longer maintained
    这个坑是真的踩的扎心了，折腾了我很久，网上百度也是五花八门的答案，后来机缘巧合之下
    让我找到了答案，原来只是网络慢，一直在下载中，所以，老铁们，遇到这个问题，丢在那里，
    什么都不要做，过会就好了

  4. 以上完成以后，通过命令cd blog道blog文件夹目录下，输入：hexo server
    提示： Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.
    这个时候博客就搭建完成了，就这么几个步骤，我竟然花了一整天的时间才研究一个入门，泪奔
    不过总归是完成了，还是很有成就感的，撒花
