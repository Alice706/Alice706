nodejs

  • v8 是非常快的js引擎，处理js执行运行的速度非常高
  • 定义：运行在服务端的js
    1. 引入 required 模块：我们可以使用 require 指令来载入 Node.js 模块。
  1. 创建服务器：服务器可以监听客户端的请求，类似于 Apache 、Nginx 等 HTTP 服务器。
  2. 接收请求与响应请求 服务器很容易创建，客户端可以使用浏览器或终端发送 HTTP 请求，服务器接收请求后返回响应数据。
