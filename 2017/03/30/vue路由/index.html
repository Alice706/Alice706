vue路由，路由传参

vue-router

  • 路由哪里去
  • 前端路由原理
    • 监视当前的锚点值，根据锚点值得不同，在指定的地方，插入不同的模板内容

      路由中的对象

    • 路由信息对象 $route
    • 路由功能对象 $router
    • 安装插件时，挂载的相关属性
      基本使用
    • 引入该对象 improt VueRouter from 'vue-router';
    • 安装插件 Vue.use(VueRouter)
    • 配置路由（创建一个关于当前实例的路由对象）
      let router = new VueRouter(构造选项)
      • 构造选项
        {routes:[{锚点值，显示什么}]}
    • 留坑
      <router-view></router-view>

router-link
<router-link to="/xxx"></router-view>//这里写的是一个具体的地址

命名路由

  1. 去哪里：<router-link :to="{name:'xxx'}"></router-link>
  2. 导航，配置路由：routes:[{name:'xxx',path:'/header',component:Header}]

  3. 去了干嘛：’获取路由相关参数’

    路由传参

  4. 通过查询字符串的方式 query

    • this.id = this.$route.query.id //详情页中 detial中
    • <router-link :to="{name:'diao',query:{id:hero.id}}"></router-link> //要渲染的页面中app.vue
    • {name:'diao',path:'/detial',component:Detial}, //main.js中

  5. 使用path的方式

    • this.id = this.$route.params.id //详情页中 detial中
    • <router-link :to="{name:'diao',params:{id:hero.id}}"></router-link> //要渲染的页面中app.vue
    • {name:'diao',path:'/detial/:id',component:Detial}, //main.js中
