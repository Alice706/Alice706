对象、面向对象、原型

对象

  • 无序的键值对的集合

  • 创建对象的方式

    1. 字面量：var obj = { } //缺点：大量使用时，代码的复用性太低
    2. 构造函数：var obj = new Object();//代码复用性更低
    3. 自定义构造函数：function Obj() { } var obj1 = new Obj();var obj2 = new Obj();
      • 执行过程：
      1. 创建一个对象
      2. 让this 指向这个对象
      3. 执行构造函数
      4. 返回这个对象
    4. 实例：对象，实例化：创建对象的过程
    5. 静态方法和实例方法

    • 静态方法：函数名.方法名

    • 实例方法：对象名.方法名

    • 如何给对象添加属性和方法
      1. 添加到原型上
      2. 让原型指向这个对象

  • 注：对象的返回值是引用类型时，返回的结果就是引用类型

原型

  • 概念：每个构造函数在创建出来的时候，系统会自动给这个构造函数创建并关联一个空对象，这个对象就是原型
  • 在原型中定义的方法和属性，会被这个构造函数创建出来的对象共享

面向对象

  • 面向对象就是面向过程的封装
  • 面向对象是一种编程思想
  • 原型链：对象有原型，原型本身又是一个对象，所以原型也有原型
  • 三大特性
    • 封装（可拓展性）对象对属性和方法的封装
    • 继承 （重用性）一个对象想拥有另外一个对象的属性和方法
    • 多态（可维护，可拓展）在Js中没有多态

    经典继承

  1. 简单混入 for … in(缺点：复用性低)
    var obj = {
    name : ‘nacy’;
    }
    var obj1 = {};
    for (var k in obj){
    obj1[k] = obj[k]
    }

    2. 原型混入
    - for … in 添加到原型中
    - 替换原型

    var obj = {
    name : “nacy”
    }
    function Person () {}
    var person = new Person();
    p.prototype = obj;

    3. Object.create()

    var obj = {
    name:’nacy’
    }
    var obj1 = Object.create(obj);//好用，es6的新语法，有兼容性
