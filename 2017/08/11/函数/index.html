创建函数的几种方式

  1. 字面量
    • function fn () { }
  2. 函数表达式
    • var fn =function () { }
  3. 构造函数
    • var fn = new Function ();
    • var fn = new Function(‘hello’);
  4. 自定义构造函数
    • function Fn () { }
      var fn = new Fn();
    callee和caller
    • callee:argument对象所在的函数
      作用：在函数内部指向函数自身
      function fn (1,2,3) {
      arguments.callee;//结果：这个函数体
      arguments.callee();//递归
      }
    • caller呼叫者
      作用：显示该函数的调用者
      如果fn()在全局调用，返回结果为null
      function fn (a,b) {
      console.log(fn.caller);
      }
      function fn1 ( ) {
      fn();
      }

返回：function fn1 () {
fn();
}
